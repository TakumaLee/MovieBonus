## SEO 優化建議

### 1. 內容優化
```html
<!-- 首頁應包含更多描述性內容 -->
<h1>特典速報 パルパル - 台灣最完整的電影特典追蹤平台</h1>
<p>即時更新威秀影城、秀泰影城等各大戲院的電影特典資訊...</p>
```

### 2. 圖片優化
- 添加 og-image.jpg 到 public 資料夾
- 所有圖片都要有 alt 屬性
- 使用 Next.js Image 組件優化載入

### 3. 效能優化
- 使用 Lighthouse 測試網站效能
- 優化首屏載入時間
- 減少 JavaScript bundle 大小

### 4. 本地 SEO
- 明確標示服務地區（台灣）
- 使用繁體中文內容
- 加入本地相關關鍵字
